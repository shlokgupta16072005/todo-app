# To-Do App

## Description
A simple web app to manage daily tasks. Users can add, delete, and mark tasks as completed. Tasks are saved using localStorage.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Persist tasks using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
# Clone the repository
git clone https://github.com/yourusername/todo-app.git

# Open the project
cd todo-app
# Open index.html in your browser
